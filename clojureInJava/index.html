What is Clojure ?

  • Clojure is a Lisp dialect for the JVM.
  • It can call Java code directly.
  • Java code can call Clojure
  • There's also ClojureCLR for the .NET platform.
  • You can run it in your browser using ClojureScript.

    Summary

    • Clojure is a Lisp dialect for the jvm
    • Clojure code can use any java library
    • You only need to include the clojure jar to use it
    • Clojure projects can use Leiningen as a build tool
    • Clojure code can be precompiled as a "java-class" to be called from java
    • Clojure is fun!
